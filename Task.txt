Тема лабораторной работы: Графовая база данных в MS SQL Server

Вариант задания: Авиаперелёты — Аэропорты, рейсы, авиакомпании
                 (стыковки и время ожидания)

Описание предметной области:
---------------------------------------------------------------------------
Необходимо создать графовую базу данных для хранения информации об
авиационной маршрутной сети. База данных моделирует:

1. Аэропорты (узлы Airport) — международные и внутренние аэропорты
   с кодами IATA, наименованием, городом и временной зоной.

2. Авиакомпании (узлы Airline) — авиаперевозчики, их страна и год основания.

3. Города (узлы City) — населённые пункты, в которых расположены аэропорты.

4. Рейсы (рёбра Flight: Airport → Airport) — прямые авиарейсы между
   аэропортами с номером рейса, продолжительностью (мин), расстоянием (км)
   и стоимостью (руб).

5. Эксплуатирует (рёбра Operates: Airline → Airport) — связь авиакомпании
   с аэропортом с указанием года начала полётов, признака хаба и числа
   рейсов в неделю.

6. Расположен в городе (рёбра LocatedIn: Airport → City) — связь аэропорта
   с городом с расстоянием до центра и способами добраться.

7. Стыковка (рёбра Connection: Airport → Airport) — возможность пересадки
   в данном аэропорту с минимальным временем ожидания, признаком смены
   терминала и типом стыковки (внутренняя / международная).

Задачи, решаемые с помощью графовой базы:
- Найти все прямые рейсы из заданного аэропорта.
- Найти маршруты с одной или несколькими пересадками.
- Выяснить, какие авиакомпании оперируют в аэропортах конкретного города.
- Найти кратчайший маршрут (по числу рейсов) между двумя аэропортами.
- Определить все аэропорты, достижимые из заданной точки за N рейсов.
